This site is a lightweight documentation workspace for lab notes, experiment
instructions, and supporting reference material.

## What belongs here

- Course overview and schedules
- Experiment setup instructions
- Submission requirements
- Notes that help students reproduce the lab environment

## Suggested structure

Keep each experiment in its own page and focus on the information students need
to complete the work:

1. Objective
2. Environment and dependencies
3. Steps
4. Expected output
5. Submission checklist
